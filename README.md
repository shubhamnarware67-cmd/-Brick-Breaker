# 🎮 Brick Breaker Pro

A browser-based **Brick Breaker game** built using **HTML, CSS, and JavaScript** in a single page.
The game includes levels, power-ups, pause functionality, lives system, and high score tracking.

---

## 📌 Project Overview

Brick Breaker Pro is a simple web game where the player controls a paddle to bounce a ball and break bricks.
The objective is to destroy all bricks while preventing the ball from falling below the paddle.

This project demonstrates fundamental **game development concepts using JavaScript and HTML5 Canvas**.

---

## 🚀 Features

* 🏠 Start menu
* 🎚 Multiple levels
* ⚡ Power-ups (paddle expansion)
* ❤️ Lives system
* ⏸ Pause functionality
* 🏆 High score tracking using LocalStorage
* 🎮 Keyboard controls
* 📱 Mobile touch support
* 🎨 Simple and clean UI

---

## 🛠 Technologies Used

* **HTML5** – Page structure
* **CSS3** – Styling and layout
* **JavaScript (ES6)** – Game logic
* **HTML5 Canvas API** – Rendering graphics
* **LocalStorage API** – Saving high scores

---

## 🎮 How to Play

1. Open the game in your browser.
2. Click **Start Game**.
3. Use the **Left Arrow (⬅️)** and **Right Arrow (➡️)** keys to move the paddle.
4. Break all bricks using the ball.
5. Catch falling **power-ups** to increase paddle size.
6. Avoid missing the ball.

### Controls

| Action         | Key                    |
| -------------- | ---------------------- |
| Move Paddle    | ⬅️ ➡️ Arrow Keys       |
| Pause Game     | P key                  |
| Mobile Control | Slide finger on screen |
